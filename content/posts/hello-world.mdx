# 歡迎來到我的部落格

這是我使用Velite建立的第一篇Markdown部落格文章。

## 關於Velite

Velite是一個強大的工具，可以將Markdown文件轉換為結構化的資料，非常適合建立靜態部落格。

## 主要特色

- **簡單易用**: 只需將Markdown文件放入content目錄
- **型別安全**: 使用Zod schema進行驗證
- **快速**: 在建置時處理內容

## 程式碼範例

```javascript
// 這是一個簡單的JavaScript範例
function greet(name) {
    return `你好，${name}！`;
}

console.log(greet('世界'));
```
